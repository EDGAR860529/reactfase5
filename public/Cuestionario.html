Cuestionario de Box Model

Pregunta 1:

¿Cuál de las siguientes propiedades define el tamaño total de un elemento en el modelo de caja?

a) Width
b) Margin
c) Padding
d) Height

Pregunta 2:

¿Qué propiedad se utiliza para definir el espacio entre los bordes de un elemento y sus contenidos?

a) Margin
b) Width
c) Padding
d) Border

Pregunta 3:

¿Cuál de las siguientes afirmaciones describe correctamente el modelo de caja?

a) El modelo de caja solo se aplica a elementos de bloque.
b) El modelo de caja se aplica tanto a elementos de bloque como a elementos en línea.
c) El modelo de caja solo se aplica a elementos en línea.
d) El modelo de caja no se utiliza en CSS.

Pregunta 4:

¿Cuál de las siguientes propiedades CSS se utiliza para establecer el ancho de un borde de un elemento?

a) margin
b) padding
c) width
d) border-width

Pregunta 5:

¿Cuál es el orden correcto de las propiedades del modelo de caja en CSS?

a) margin, border, padding, content
b) padding, border, margin, content
c) border, padding, margin, content
d) content, margin, padding, border
